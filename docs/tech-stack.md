# 勤怠・シフト管理 SaaS — 技術スタック（MVP）

## サービス名
**ロスタ (rosuta.com)**

---

## 1. フロントエンド (UI/UX)

| 項目 | 技術 / ライブラリ | 用途 |
|------|-----------------|------|
| フレームワーク | **Next.js / React** | SPA構築、ページルーティング、SSR対応 |
| スタイリング | **Tailwind CSS** | コンポーネントのスタイリング |
| カレンダー表示 | **FullCalendar / React Big Calendar** | シフト・勤務状況カレンダー表示 |
| 状態管理 | **React Query / Zustand** | サーバー状態管理、キャッシュ |
| 日付操作 | **date-fns / Luxon** | 日付計算・フォーマット |
| モーダル | **Radix UI / Headless UI** | 打刻・時刻選択モーダル |
| アイコン | **Lucide / Heroicons** | UIアイコン表示 |
| フォーム管理 | **React Hook Form** | 打刻・シフト入力フォーム管理 |
| バリデーション | **Zod / Yup** | 入力値チェック |

> 💡 モバイル対応：レスポンシブデザインでスマホ打刻・カレンダー確認を想定

---

## 2. バックエンド (API / 認証 / DB操作)

| 項目 | 技術 / ライブラリ | 用途 |
|------|-----------------|------|
| バックエンド | **Supabase / PostgreSQL** | DB + 認証 + API (REST/GraphQL) |
| 認証 | **Supabase Auth (Email/Password)** | スタッフ・管理者ログイン |
| API通信 | **Axios / Fetch** | フロント → バックエンド通信 |
| サーバー状態判定 | **Supabase Functions / Edge Functions** | 打刻ステータス判定、承認フロー処理 |
| バックアップ | **Supabase Backups** | データ保全 |


---

## 3. データベース

| 項目 | 技術 / ライブラリ | 用途 |
|------|-----------------|------|
| インデックス | B-Tree / GIN | クエリ高速化 (スタッフ別・店舗別検索) |

---

## 4. インフラ / ホスティング

| 項目 | 技術 / サービス | 用途 |
|------|-----------------|------|
| フロントホスティング | **Vercel** | Next.js SSR/静的ホスティング |
| バックエンド / DB | **Supabase Cloud** | PostgreSQL + Auth + Functions |
| ドメイン | **Cloudflare / Google Domains** | SSL・DNS管理 |
| CDN | Supabase / Vercel | 静的資産配信、キャッシュ高速化 |

---

## 5. 開発ツール / CI/CD

| 項目 | 技術 / ツール | 用途 |
|------|-----------------|------|
| バージョン管理 | **Git / GitHub** | ソース管理・プルリク管理 |

## 7. 補足

- **複数店舗対応**：フロント・バックエンド・DBすべてに反映  
- **承認設定**：店舗ごとにON/OFF可能  
- **打刻履歴**：出勤・退勤・休憩開始・休憩終了 + 時刻選択方法・実打刻時刻・店舗情報を保持  
- **カレンダー表示**：スタッフ・管理者で表示切替、過去履歴・当日ステータスも可視化
